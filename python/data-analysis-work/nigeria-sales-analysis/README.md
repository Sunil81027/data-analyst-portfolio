Nigerian Electronics Sales Data Analysis

Project Overview

This project focuses on analyzing a Nigerian electronics sales dataset to extract meaningful business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and visualization to understand sales patterns, product performance, and potential business opportunities.

The goal of this project is to demonstrate practical Data Analysis skills using Python and commonly used data analysis libraries.

Objectives

- Clean and preprocess a messy sales dataset
- Perform Exploratory Data Analysis (EDA)
- Identify sales trends and patterns
- Generate business insights from the data
- Visualize important findings using graphs


Tools & Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook


Project Files

The repository contains the following files:

- "nigeria_messy_sales_dataset.csv" → Raw dataset used for analysis
- "Nigerian_sales_dataset.ipynb" → Python notebook containing data cleaning and analysis code
- "Sunil_Kumar_Nigerian_sales_dataset.pdf" → Final project report with insights and visualizations


Data Analysis Steps

The analysis process includes the following steps:

1. Data Loading
   
   - Import dataset using Pandas

2. Data Cleaning
   
   - Handling missing values
   - Removing duplicates
   - Correcting data formats

3. Exploratory Data Analysis (EDA)
   
   - Understanding dataset structure
   - Analyzing sales distribution
   - Identifying top-selling products

4. Data Visualization
   
   - Sales trends using charts
   - Product performance comparison
   - Graphical insights using Matplotlib & Seaborn


Key Insights

Some important insights derived from the dataset include:

- Identification of top-performing electronic products
- Sales trends across different product categories
- Insights into potential areas for improving business sales


Skills Demonstrated

This project demonstrates the following data analysis skills:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Python for Data Analysis
- Business Insight Generation
